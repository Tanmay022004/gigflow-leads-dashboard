# GigFlow – Smart Leads Dashboard

A full-stack CRM-style leads management system built for internship assignment evaluation.

---

## 🚀 Live Links
  
Backend: https://gigflow-leads-dashboard.onrender.com

---

## 📌 Features

- User Authentication (JWT)
- Role-Based Access Control (Admin/User)
- Create, Read, Update, Delete Leads
- Search leads by name/email
- Filter by status & source
- Sort by latest/oldest
- Pagination support
- Export leads to CSV
- Protected API routes

---

## 🧱 Tech Stack

### Frontend
- React (Vite)
- TypeScript
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication
